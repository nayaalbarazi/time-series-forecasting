# time-series-forecasting
Forecasting future stocking demand using historical time series data

## dataset 
This dataset includes historical stock data for Apple Inc. from 1/1/2020 to 31/12/2023

## Tools
- Python
- Jupyter Notebook
- Pandas & Numpy
- Matplotlib, Seaborn
- Statsmodels / Scikit-learn

## Workflow
- Time parsing and datetime indexing
- Data visualizationa and trend analysis
- Handling missing values and outliers
- Decomposition
- Model training using ARIMA


##Results
- Best model performance: RMSE: 210.4
- Forecast for next 30 days visualized
